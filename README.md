# 📌 Project Overview

This project is an intelligent **Sentiment Analysis System** built using **Natural Language Processing (NLP)** and **Machine Learning** techniques.

The system processes textual reviews, cleans and transforms the text into meaningful numerical features using **TF-IDF Vectorization**, and predicts sentiment using a **Logistic Regression** model.

The project demonstrates the complete NLP workflow including:

* Text preprocessing
* Data cleaning
* Feature extraction
* Model training
* Sentiment prediction
* Accuracy evaluation

# 🔍 NLP Workflow

## Step 1: Data Collection

* Load dataset containing text reviews.

## Step 2: Text Preprocessing

* Convert text to lowercase
* Remove HTML tags
* Remove punctuation and special characters

## Step 3: NLP Processing

* Tokenization
* Part-of-Speech tagging
* Text normalization

## Step 4: Feature Extraction

* Convert text into TF-IDF vectors.

## Step 5: Model Training

* Train Logistic Regression classifier.

## Step 6: Evaluation

* Predict sentiments
* Calculate model accuracy
